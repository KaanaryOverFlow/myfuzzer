 # requirements
 - python3
 - android emulator or read devices
 - adb tool

# usage
- connect to android with adb tool
- set device id to [ayarlar.py](https://github.com/MucahitSaratar/myfuzzer/blob/main/ayarlar.py)
- $ ```python3 fuzzer.py```
